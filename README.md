# Pharma Log Anomaly Detector

AI-powered log anomaly detection and validation support platform for pharmaceutical computer systems.

## Overview

This project aims to automate log analysis for pharmaceutical computer systems by detecting anomalies using machine learning techniques and providing validation-friendly reporting.

## Planned Features

- User Authentication
- CSV Log Upload
- Log Parsing
- AI-Based Anomaly Detection
- Risk Classification
- Root Cause Suggestions
- Dashboard & Analytics
- Audit Trail
- PDF Report Generation
- Feedback Management

## Technology Stack

- ASP.NET Core MVC (.NET 10)
- SQL Server
- Entity Framework Core
- Python (paandas, NumPy, scikit-learn)
- Bootstrap 5

## Project Status

Environment setup and architecture design in progress.
